# TradeHub - Peer-to-Peer Marketplace App

TradeHub is a **Flutter/Kotlin-based peer-to-peer marketplace app** that enables users to buy and sell second-hand products efficiently. The app provides secure transactions, product discovery, in-app chat, and ratings to build trust among users.

---

## 🛠 Features

- **Product Listing**: Users can view products with images, title, price, description, and seller info.  
- **Search & Filter**: Quickly find products using the search bar.  
- **Add Product**: Users can add their own products via a simple form.  
- **Product Details**: Detailed view with price, description, rating, seller info, and “Buy Now” button.  
- **In-App Chat**: Direct messaging between buyers and sellers.  
- **Ratings & Reviews**: Display product ratings and seller trust indicators.  
- **Mock Payment**: Simulated payment system with confirmation.  
- **Offline Caching**: Basic offline support for previously loaded products.  
- **Clean UI**: Material3 design for modern look and feel.
